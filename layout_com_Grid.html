Este website explora a combinação do CSS Grid e Flexbox na criação de um layout.
Na disposição dos elementos (header, nav, main, aside e footer) usa-se o CSS Grid, recorrendo à disposição de grid-areas numa grid-template-areas.
Por outro lado é usado o CSS Flexbox para o texto no centro de cada contentor.
